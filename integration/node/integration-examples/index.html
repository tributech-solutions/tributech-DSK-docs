    Integration Examples¶

    The simplest way to integrate our APIs is through the offered nuget package.

    For easy integration of the Data API and Trust API we also offer a public GitHub repository which contains our API clients and code-samples showcasing how the APIs can be used.

    If you do not use C# you can generate your own API Client using any one of the publicly available OpenAPI code generators, such as the OpenAPI Generator.
